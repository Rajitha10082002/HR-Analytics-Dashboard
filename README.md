# HR Analytics Dashboard

## Project Overview

This project is an HR Analytics Dashboard created using Power BI.



## Features

* Employee Count
* Attrition Analysis
* Attrition Rate
* Average Salary
* Attrition by Education
* Attrition by Job Role
* Attrition by Age
* Attrition by Salary Slab

## Tools Used

* Power BI
* Excel / CSV
* Data Visualization
